# UMS
Under Graduation Project. Designed a robust, highly flexible, high-performance student and course management system with student, staff, and faculty roles, IDs, and functions. Built a role based log-in screen and user interface. Programmed modules for new course entry, semester planning / course setting with a mouse listener, user settings, semester settings, and department-based display of subjects, students, and faculties. Developed an advanced search option using Intelligent Search functions. Implemented registration and account management. Tools include Java SDK 1.8, Eclipse, JDBC, MySQL, and MVC.
